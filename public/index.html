Button Simulator

Here, you can create your own buttons and copy over the css to your own personal projects and websites without the mess of creating and deleting css properties over and over again!

At the top of the website, you can select the follwoing 3 buttons
- Home Page
- Pick Page
- Database Page

When you select the pick page, you can select the button with your desired styling.
- Back undo's your previous styling choice.
- Refresh will re-load new styles for the current css property.
- Reset will undo all of your styles applied so far.
- The skip button will skip the current css styling property.
